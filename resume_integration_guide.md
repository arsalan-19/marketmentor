💡 Section 3: The 30-Second Interview Pitch

When a recruiter or hiring manager asks: "Tell me about a project you're proud of," deliver this structured response:

"I designed MarketMentor, an AI-powered NSE swing trading assistant for Android.

The core problem I wanted to solve was that over 70% of retail investors in India lose money in their first year due to high-risk, speculative day trading and emotional bias. To solve this, I designed a strict delivery-only ($T+1$ holding) sandbox that prohibits all intraday leverage and derivative options.

I wrote the PRD, mapped the user journeys, and compiled a hybrid data-processing system that batches heavy fundamental analysis pre-market, running light technical calculations on the client's phone to keep server costs near zero.

To make developer and stakeholder alignment seamless, I also built a fully functional web companion prototype hosted on GitHub Pages. It has interactive jargon tooltips and an intraday lock system to demonstrate how the app protects capital in real time."
